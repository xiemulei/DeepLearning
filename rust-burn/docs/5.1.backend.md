# 后端 (Backend)

Burn 中的几乎所有内容都基于 `Backend` trait，它使您能够使用不同的实现运行张量操作，而无需修改代码。虽然后端不一定具有自动微分功能，但 `AutodiffBackend` trait 指定了何时需要自动微分。这个 trait 不仅抽象了操作，还抽象了张量、设备和元素类型，为每个后端提供了它们所需的灵活性。值得注意的是，该 trait 假设使用即时执行模式，因为 Burn 完全支持动态图。但是，我们可能会创建另一个 API 来帮助集成基于图的后端，而不需要对用户代码进行任何更改。

用户不需要直接使用后端 trait 方法，因为它主要是为后端开发者设计的，而不是为 Burn 用户。因此，大多数 Burn 用户级 API 都是跨后端通用的。这种方法帮助用户通过适当的自动补全和文档更自然地发现 API。
