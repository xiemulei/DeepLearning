# 概览

欢迎使用 Burn 书籍 👋

本书将帮助您开始使用 Burn 深度学习框架，无论您是高级用户还是初学者。我们为您精心准备了一些章节：

## 章节内容

### [基本工作流程：从训练到推理](./basic-workflow)
我们将从基础开始，指导您完成整个工作流程，从训练模型到部署模型进行推理。本节为您掌握 Burn 奠定基础。

### [构建模块](./building-blocks)
深入了解 Burn 的核心组件，了解它们如何协同工作。这些知识为更高级的使用和自定义奠定了基础。

### [性能 - 最佳实践](./performance/good-practices/)
编写模型和训练代码的技巧，充分利用硬件资源，同时避免可能降低执行速度的常见陷阱。

### [自定义训练循环](./custom-training-loop.html)
获得自定义训练循环的能力，微调您的模型以满足特定需求。本节让您充分利用 Burn 的灵活性。

### [保存和加载模型](./saving-and-loading.html)
学习如何轻松保存和加载训练好的模型。

### [导入模型](./import)
学习如何导入 ONNX 和 PyTorch 模型，扩展您与其他深度学习生态系统的兼容性。

### [模型和预训练权重](./models-and-pretrained-weights.html)
使用现成的模型和预训练权重快速开始。

### [高级主题](./advanced)
最后，探索高级主题，充分发挥 Burn 的能力。本节面向那些想要突破 Burn 可能性边界的人。

## 学习前提

在整本书中，我们假设您对深度学习概念有基本了解，但在适当时我们可能会参考其他材料。
